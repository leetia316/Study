- 指令和数据在内存中是没有区别的

## 指令

### 指令执行过程

- CUP从 当前的 CS 和 IP 指向的内存单元读取指令，然后将读取的指令存入指令缓冲器
- IP = IP + 所读指令的字节数，从而指向下一条指令
- 执行指令缓冲器中的内容， 回到第一步

### 转移指令(jmp)

> 改变 CS IP 寄存器的 汇编指令

```assembly
jmp 2AE3:0003 ; CS = 2AE3H , IP = 0003H
jmp 0003 ; IP = 0003H
```

### 移动指令(mov)

> 将数据从一个地方移动到另一个内存中

```assembly
mov ax, 1000 ; 将 1000H 存入 ax 寄存器中， ax = 1000H
```

### 减法指令(sub)

```assembly
sub ax, ax ; ax = ax - ax
```
