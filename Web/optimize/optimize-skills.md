### 缩小资源

> 缩小是从基于文本的资源中去除所有空白和非必要字符的全过程

#### 工具

```bash
npm install -g minifier html-minify
```

#### 目标

- css
- js
- html
  - html的空白对布局可能会产生影响所以特殊处理

### 服务器压缩

> 合并文件，减小文件体积

#### 工具

```bash
npm install compression 
```

#### 目标

- 文本

### 图像压缩

> 丢弃图像中不必要的数据，并且不明显影响图像视觉质量
